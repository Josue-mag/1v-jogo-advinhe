# Jogo de Adivinhação

Este projeto é um **protótipo de jogo** desenvolvido com o objetivo de **treinar JavaScript**, além de praticar conceitos básicos de **HTML e CSS**.

O jogo foi criado como parte do meu aprendizado em lógica de programação, manipulação do DOM e estruturação de interfaces simples para web.

##  Status do Projeto
- Em desenvolvimento (protótipo)

##  Tecnologias Utilizadas
- HTML
- CSS
- JavaScript

##  Próximos Passos
Este projeto **receberá atualizações futuramente**, com melhorias na interface, novas funcionalidades e ajustes no código conforme a evolução dos estudos.
